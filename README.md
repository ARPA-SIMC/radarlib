Radarlib
===============================================================


Radarlib is a library to manage radar volumes and products 
written in ODIMH5 format (v2.0) as defined by OPERA (EUMETNET 
project).


Documentation
-------------

The documentation available is generated by Doxygen in the doc/html 
directory.

Software dependencies
---------------------

Radarlib should build and run on any reasonably recent Unix system, 
and has been tested on Debian based and Red Hat based distributions.
HDF5 libraries are required.
